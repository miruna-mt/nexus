📋 Sobre este escenario

Optimización de centros de datos: asignar máquinas virtuales a servidores físicos minimizando el consumo energético y respetando restricciones de capacidad y afinidad.

  • Servidores: 3 tipos (High CPU, Balanced, Storage Optimized) con diferente capacidad y consumo.
  • Máquinas virtuales: 5 VMs (web servers, database, cache, backup) con requisitos de CPU, RAM y almacenamiento.
  • Restricciones: Capacidad de servidores, afinidad (DB y cache juntos), anti-afinidad (web servers separados).

Ejemplo de resultado: El modelo asigna VMs a servidores minimizando el consumo total (vatios) y cumpliendo todas las restricciones.
